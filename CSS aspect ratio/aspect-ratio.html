/* CSS aspect-ratio is a property that allows you to set the aspect ratio of an element. The aspect ratio is the proportional relationship between the width and height of an element. */ /* The CSS aspect-ratio property specifies the ratio of the width to the height of the element's box. In the code aspect-ratio: 3/2, it means that the width of the element will be 1.5 times the height of the element. This can be used to set the aspect ratio of an element, such as an image or video, to a specific value, while allowing the height or width to adjust based on the content or container size. For example, if the height of an element is 200 pixels, then the width would be 1.5 times that, or 300 pixels. If the height changes to 300 pixels, then the width would be 450 pixels to maintain the 3:2 aspect ratio. */
